A Pig Game

One die Pig Game App

This project was built using HTML, CSS and JavaScript. Click the image below to see how it works!!

This game can be played by two players each taking turn after the rolling of a die. Click the ROLL DICE button to display the value of the die and click on HOLD to claim the value. When a 1 is shown on the die, the second player takes the turn. To make the game more fun, when a 6 is shown, the player loses all previously accumulated points. The player can claim the point after every die thrown until the maximum point is reached, to decide the winner.

Players can set the maximum point as a winning score before the game commences and if there is no winning score set, there is an automatic winning score of 100 points. Enjoy!!
