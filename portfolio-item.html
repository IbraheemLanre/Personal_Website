A Random Quote Machine

With direct twitter post

This project was build using API to get online random quotes of well known people all around the world. Click here the image below to see how it works!!

Each time you click on the New Quote button, it automatically fetches a new quote, you can as well post the quote as a tweet on your twitter account, it takes you to a new where you can put your twitter login details. Try it and see how it works!!

It was built using React and fecthing random Quote online and material-ui design as the CSS tool.
